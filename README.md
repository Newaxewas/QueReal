# QueReal

Task setting system in form of game quests. 

## Functions

- Task setting 
    - Personal
    - For friends
    - For groups
- Friend system
    - Add/remove friend
    - Search user
    - Add to favorite

## Used technologies

- .Net 6
- ASP.Net Core 6 MVC
- Entity Framework core 6
- NUnit testing framework with Moq and Fluent Assertions
